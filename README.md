# 30-Days-Data-Science-Sprint

This repository documents my 30-day intensive Data Science sprint focused on building practical skills, consistency, and a strong beginner portfolio. The goal is to complete daily DataCamp learning, mini-projects, and GitHub commits without breaking the streak.

## Goals

- Build daily coding consistency
- Improve Python, SQL, pandas, and data visualization skills
- Complete guided and self-practice projects
- Create a visible GitHub contribution streak
- Prepare for DataCamp certifications and entry-level data roles

## What this repository includes

- Daily SQL practice files
- Python notebooks
- Data cleaning and analysis mini-projects
- Data visualization exercises
- Portfolio-ready project work
- Progress notes and documentation

## Tools used

- DataCamp
- Python
- SQL
- pandas
- Matplotlib / Seaborn
- GitHub

## Sprint structure

- Days 1-7: Python foundations 
- Days 8-15: SQL foundations
- Days 16-22: Data wrangling with pandas
- Days 23-27: Data visualization and insight writing
- Days 28-30: Portfolio, review and modification
